# Tetris-Style
 
A simple Tetris game implemented in C++ and ncurses.
 
# How to build
 
Requirements:

g++

ncurses

```bash
g++ tetris-style.cpp -lncurses -o tetris

./tetris
```
 
# How to Play
 
```keyboad

    a/d: Move left/right
    s: Soft drop
    w: Rotate
    crtl-c: end the game

```
